## Running the mongo db


```bash
# development
$ docker compose-up
```

## Running the server

```bash
# go to backend directory
$ cd server

# watch mode
$ yarn start

# production mode
$ yarn build
```

## Running the client

```bash
# go to frontend directory
$ cd client

# watch mode
$ yarn start

# production mode
$ yarn build
```
